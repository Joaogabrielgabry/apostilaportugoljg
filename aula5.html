Estrutura Básica em Portugol

O esqueleto que todo programa precisa ter — do primeiro ao último caractere.

Progresso
5 / 15

Todo programa em Portugol segue uma estrutura padrão. Essa organização mantém o código claro, legível e fácil de manter — desde o primeiro exercício até projetos maiores.

// exemplo completo

exemplo.por portugol 
algoritmo "Exemplo"

var
   nome: caractere

inicio
   escreva("Digite seu nome: ")
   leia(nome)
   escreva("Olá, ", nome)

fimalgoritmo

// anatomia do programa

algoritmo

Define o nome do programa. Identifica sua finalidade para quem lê o código.

algoritmo "Cadastro"
var

Área de declaração de variáveis. O programa reserva espaço na memória aqui.

var
  idade: inteiro
inicio

Marca onde a execução começa. Todas as instruções executáveis ficam aqui.

inicio
  // instruções aqui
fimalgoritmo

Encerra o programa. Após essa linha, nenhuma instrução é executada.

fimalgoritmo // fim

Pontos Importantes

  • Todo programa começa com algoritmo
  • Variáveis ficam dentro de var
  • Execução ocorre entre inicio e fimalgoritmo
  • Organização reduz erros

Por que Importa

  • Evita confusão na leitura do código
  • Facilita manutenção futura
  • Melhora a legibilidade geral
  • Prepara para linguagens reais
🎯

Criar padrão desde o início desenvolve disciplina técnica. Esse hábito acelera a transição para linguagens profissionais como Python, Java e C.
