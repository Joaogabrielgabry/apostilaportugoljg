Calculadora Básica

Realize as quatro operações aritméticas com validação inteligente contra divisão por zero.

Progresso
14 / 15

🧮

O programa lê dois números e executa as quatro operações aritméticas de uma só vez, exibindo todos os resultados.

Uma verificação prévia garante que a divisão por zero seja tratada antes de qualquer cálculo, evitando erros em tempo de execução.

// requisitos do sistema

01

Ler dois números do tipo real

02

Verificar se o divisor é diferente de zero

03

Calcular as quatro operações

04

Exibir todos os resultados ou aviso de erro

// operações realizadas

Exemplo com número1 = 10 e numero2 = 2

➕ Soma 10 + 2 = 12
➖ Subtração 10 - 2 = 8
✖️ Multiplicação 10 * 2 = 20
➗ Divisão 10 / 2 = 5
⚠️ Se numero2 fosse 0 — bloco bloqueado numero2 != 0
Operador de validação utilizado != (diferente de)

// exemplo de solução

calculadora.por portugol 
programa{

   funcao inicio(){

      real numero1
      real numero2
      real Soma
      real Subtracao
      real Multiplicacao
      real Divisao

      escreva("Digite um número: ")
      leia(numero1)

      escreva("\nDigite um segundo número: ")
      leia(numero2)

      se(numero2 != 0){

         Soma           = numero1 + numero2
         Subtracao      = numero1 - numero2
         Multiplicacao  = numero1 * numero2
         Divisao        = numero1 / numero2

         escreva("\nSoma: ",          Soma)
         escreva("\nSubtração: ",    Subtracao)
         escreva("\nMultiplicação: ", Multiplicacao)
         escreva("\nDivisão: ",       Divisao)

      } senao {

         escreva("Não é possível dividir por zero.")

      }

   }

}

// objetivos do exercício

🔢

Usar o tipo real para números com decimais

➕

Aplicar os quatro operadores aritméticos

🛡️

Validar dados antes de processar

⚖️

Usar o operador relacional != (diferente)

🔄 Evolução Natural

Esta calculadora trata bem a divisão por zero, mas ainda calcula tudo de uma vez. O próximo passo é dar ao usuário a escolha da operação — usando um menu interativo com escolha-caso.

Assim o programa se torna uma calculadora de verdade: o usuário seleciona o que quer calcular em vez de receber tudo ao mesmo tempo.
