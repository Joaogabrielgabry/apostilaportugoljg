Aula 07 · Módulo I

Entrada e Saída

A ponte entre o usuário e o programa — sem interação, o algoritmo fica fechado e sem utilidade prática.

Progresso
7 / 15

Todo programa precisa se comunicar com o usuário. Essa comunicação ocorre por meio da entrada e da saída de dados — a base de qualquer sistema interativo.

// comandos principais

⌨️
leia()
entrada de dados

Recebe informações do usuário e armazena em uma variável. O programa pausa e aguarda a digitação.

leia(variavel)
🖥️
escreva()
saída de dados

Exibe informações na tela para o usuário. Pode mostrar textos, variáveis ou combinações.

escreva("texto", variavel)

// entrada de dados

entrada.por portugol 
var
   idade: inteiro

escreva("Digite sua idade: ")
leia(idade)    // valor armazenado na variável

// lendo vários dados

multiplos-dados.por portugol 
var
   nome   : caractere
   salario: real

escreva("Digite seu nome: ")
leia(nome)

escreva("Digite seu salário: ")
leia(salario)

// saída de dados

saida.por portugol 
escreva("Idade informada: ", idade)
escreva("Nome: ", nome)
escreva("Salário: ", salario)

// exibindo resultados de cálculos

calculo.por portugol 
var
   a, b, soma: inteiro

a    <- 4
b    <- 6
soma <- a + b

escreva("Resultado da soma: ", soma)  // exibe 10

// entrada → processamento → saída completo

fluxo-completo.por portugol 
var
   n1, n2, media: real

// ENTRADA
escreva("Digite a primeira nota: ")
leia(n1)

escreva("Digite a segunda nota: ")
leia(n2)

// PROCESSAMENTO
media <- (n1 + n2) / 2

// SAÍDA
escreva("Média final: ", media)
