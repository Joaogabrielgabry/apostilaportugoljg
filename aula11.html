Cofrinho do Tempo

Calcule o tempo total de vida acumulado por um grupo de pessoas.

Progresso
11 / 15

⏳

O objetivo é calcular o total de anos vividos por um grupo. Para isso, o sistema deve somar as idades de todas as pessoas.

Uma forma simples de visualizar o tempo acumulado de experiência coletiva.

// requisitos do sistema

01

Ler a idade de cinco colegas

02

Ler sua própria idade

03

Somar todas as idades

04

Mostrar o total na tela

// exemplo de acumulação

// exemplo de solução

cofrinho-tempo.por portugol 
programa{

   funcao inicio(){

      inteiro colega1
      inteiro colega2
      inteiro colega3
      inteiro colega4
      inteiro colega5
      inteiro minha_idade
      inteiro tempovida

      escreva("Qual é a idade do colega 1? ")
      leia(colega1)

      escreva("\nQual é a idade do colega 2? ")
      leia(colega2)

      escreva("\nQual é a idade do colega 3? ")
      leia(colega3)

      escreva("\nQual é a idade do colega 4? ")
      leia(colega4)

      escreva("\nQual é a idade do colega 5? ")
      leia(colega5)

      escreva("\nQual é a minha idade? ")
      leia(minha_idade)

      tempovida = colega1 + colega2 + colega3 +
                  colega4 + colega5 + minha_idade

      escreva("\nO tempo total de vida do grupo é: ", tempovida)

   }

}
