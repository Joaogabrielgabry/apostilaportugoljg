O laço para permite executar um bloco de código repetidamente, com controle preciso sobre quando começa, quando para e como avança.

Com ele, o que antes exigia várias linhas repetidas se torna um trecho compacto e escalável.

Como o laço funciona passo a passo

🔢

Inicialização

Define o valor inicial do contador antes do primeiro ciclo

i = 0
✅

Condição

Verificada antes de cada repetição — se falsa, o loop encerra

i < 20
⬆️

Incremento

Executado ao final de cada ciclo para avançar o contador

i++
⚙️

Bloco executado

O código dentro do laço é repetido a cada ciclo válido

escreva("*")
Saída gerada (20 vezes) ********************

// exemplo de solução

central-tempo.por portugol 
programa{

   funcao inicio(){

      inteiro i, AnoNascimento, AnoAtual, idade

      // moldura superior
      para(i=0; i<20; i++)
         escreva("*")

      escreva("\nDigite o ano de nascimento: ")
      leia(AnoNascimento)

      escreva("Digite o ano atual: ")
      leia(AnoAtual)

      // moldura inferior
      para(i=0; i<20; i++)
         escreva("*")

      idade = AnoAtual - AnoNascimento

      escreva("\nIdade calculada: ", idade)

   }

}
