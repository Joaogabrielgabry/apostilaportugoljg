// tipos básicos

inteiro
Números inteiros
quantidade: inteiro
Sem casas decimais. Ex: 0, 5, -3, 100
real
Números decimais
altura: real
Com casas decimais. Ex: 1.75, 9.99, -0.5
caractere
Texto / string
nome: caractere
Letras e palavras entre aspas. Ex: "Ana"
logico
Verdadeiro / Falso
aprovado: logico
Apenas dois valores: verdadeiro ou falso
