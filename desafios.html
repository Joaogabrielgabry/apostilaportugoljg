03

Par ou Ímpar

Resto (%) se / senao Lógica
▼
🎯

Objetivo: Usar o operador de resto da divisão (%) combinado com estrutura condicional.

Peça um número e informe se ele é par ou ímpar.

💡

Use o operador % (resto da divisão). Se numero % 2 == 0, o número é par. Caso contrário, é ímpar.

Lógica do exercício
lógica pseudocódigo 
se (numero % 2 == 0) {
   escreva("Par")
} senao {
   escreva("Ímpar")
}
Saída esperada (exemplo)
Digite um número: 7 O número 7 é Ímpar

07

Desconto de Compra

Porcentagem se / senao Real
▼
🎯

Objetivo: Combinar condicional com cálculo de porcentagem para aplicar desconto.

Peça o valor de uma compra. Se ultrapassar R$ 100, aplique 10% de desconto e mostre o valor final.

💡

Desconto de 10% = multiplicar por 0.10. Valor final = valor - (valor * 0.10)

Regras
valor > 100 Aplicar 10% de desconto → mostrar valor com desconto
senao Mostrar o valor normal sem desconto
Saída esperada (exemplo)
Digite o valor da compra: 150.00 Desconto aplicado: R$ 15.0 Valor final: R$ 135.0
08

Maior entre Três Números

⭐ Desafio se aninhado Lógica avançada
▼
⭐

Desafio! Este exercício é mais difícil — você vai precisar combinar vários se aninhados para comparar três valores ao mesmo tempo.

🎯

Objetivo: Praticar estruturas condicionais aninhadas para comparar três variáveis.

Peça três números ao usuário e informe qual deles é o maior.

💡

Estratégia: compare A com B primeiro. Se A for maior, compare A com C. Se B for maior, compare B com C. Assim você encontra o maior dos três!

Saída esperada (exemplo)
Digite o número A: 42 Digite o número B: 17 Digite o número C: 99 O maior número é: 99
