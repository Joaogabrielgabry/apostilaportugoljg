📝 Exercício 1 — Nome e Idade
BÁSICO

Crie um programa que capture e exiba informações pessoais do usuário.

  • Leia o nome do usuário
  • Leia a idade do usuário
  • Mostre ambos na tela
// EXEMPLO DE SAÍDA
Nome: Carlos
Idade: 20
📝 Exercício 2 — Soma de Dois Inteiros
BÁSICO

Desenvolva um programa que realize uma operação matemática simples entre dois números.

  • Leia dois números inteiros
  • Calcule a soma dos dois valores
  • Mostre o resultado na tela
// EXEMPLO DE INTERAÇÃO
Digite o primeiro número: 5
Digite o segundo número: 3
Resultado da soma: 8
📝 Exercício 3 — Dobro de um Número Real
BÁSICO

Implemente um programa que trabalhe com números decimais e multiplicação.

  • Leia um valor real (com casas decimais)
  • Calcule o dobro do valor informado
  • Exiba o resultado com precisão
// EXEMPLO DE INTERAÇÃO
Digite um valor: 2.5
Dobro do valor: 5.0

Dicas Estratégicas

Perguntas guia
  • O que o programa vai ler?
  • O que ele vai processar?
  • O que ele vai mostrar?
Passo a passo
  • Declare as variáveis necessárias
  • Use leia para entrada
  • Use escreva para saída
  • Realize cálculos antes de exibir
🚀 Desafio Extra

Crie um programa que leia o nome do aluno, duas notas e calcule a média final. Exiba o nome e a média calculada de forma clara e organizada.
