O Número do Universo

Combine múltiplas condições com operadores lógicos para criar programas mais inteligentes.

🌌

O programa faz ao usuário a grande pergunta do universo e aguarda uma resposta. Dependendo do que for digitado — com variações de maiúsculas, minúsculas e abreviações — uma resposta diferente é exibida.

O operador lógico ou é a chave para lidar com todas essas variações sem duplicar código.

// requisitos do sistema

03

Aceitar variações da palavra "não"

04

Tratar entradas inválidas com senao

// fluxo de decisão

O que acontece para cada resposta

✅ Usuário confirma — exibe o número do universo após 3 segundos
sim Sim S s
❌ Usuário recusa — mensagem de que não está pronto
não Não nao N
⚠️ Qualquer outra entrada — opção inválida
???

// exemplo de solução

numero-universo.por portugol 
programa{

   inclua biblioteca Util --> u

   funcao inicio(){

      cadeia universo

      escreva("Qual o significado da vida, do universo e tudo mais?\n")
      escreva("Deseja saber o número do universo?\n")
      escreva("Digite sim ou não: ")
      leia(universo)

      se(universo == "Sim" ou universo == "sim" ou
         universo == "S"   ou universo == "s"){

         limpa()
         u.aguarde(3000)
         escreva("42")

      } senao se(universo == "Não" ou universo == "não" ou
               universo == "Nao" ou universo == "nao" ou
               universo == "N"   ou universo == "n"){

         escreva("Você ainda não está preparado.")

      } senao {

         escreva("Opção inválida.")

      }

   }

}

🔄 Evolução Natural

O operador ou é apenas um dos operadores lógicos do Portugol. Nos próximos exercícios você encontrará o operador e, que exige que todas as condições sejam verdadeiras simultaneamente.

Combinando ou, e e nao, você terá controle total sobre qualquer fluxo de decisão imaginável.

💡 Dica de Ouro

O operador ou retorna verdadeiro se pelo menos uma das condições for verdadeira. Já o e exige que todas sejam verdadeiras.
