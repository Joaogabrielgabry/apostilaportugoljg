O Laboratório das Expressões

Você foi contratado para um laboratório de matemática computacional. O sistema trabalha com duas expressões matemáticas complexas: Expressão A e Expressão B.

O programa deve perguntar ao usuário qual expressão deseja calcular. Caso o usuário digite qualquer outro valor, o sistema deve informar que a opção é inválida.

// opções disponíveis

O usuário pode escolher entre

A

Calcula e exibe o resultado da Expressão A

B

Calcula e exibe o resultado da Expressão B

programa{

   funcao inicio(){

      caracter AorB

      escreva("Digite A para expressão 'A' e B para expressão 'B': ")
      leia(AorB)

      se(AorB == 'A'){

         inteiro resultadoA
         resultadoA = 2 + 3 - 5 * 8 - 4 + 354 - 521 + 7 * 66

         escreva("O resultado da expressão A é: ", resultadoA)

      } senao se(AorB == 'B'){

         inteiro resultadoB
         resultadoB = 2 + 7 * (14 - 21) + 28 * 3 * 42 + 740 - (156 + 4 + 40) * 9

         escreva("O resultado da expressão B é: ", resultadoB)

      } senao {

         escreva("Opção inválida.")

      }

   }

}

Objetivos do Exercício

  • Trabalhar entrada de dados do usuário
  • Aplicar estrutura condicional (se/senao)
  • Respeitar precedência matemática
  • Organizar o código de forma profissional
