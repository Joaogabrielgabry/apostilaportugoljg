📚 Portal de Aulas – Portugol

Projeto educacional voltado para ensino estruturado de lógica de programação com Portugol, organizado em aulas sequenciais com conteúdo teórico, exemplos práticos e quizzes interativos.

🎯 Objetivo

Capacitar alunos iniciantes a desenvolver:

Raciocínio lógico

Estruturação de algoritmos

Base sólida para linguagens como Java, C, Python e JavaScript

Autonomia na resolução de problemas

Portugol é utilizado como ferramenta didática para reduzir barreiras iniciais e acelerar aprendizado.

🧠 Conteúdo do Projeto

O portal contém:

📄 Aulas organizadas por tema

🧩 Exemplos práticos em Portugol

📝 Quiz interativo com correção automática

🎯 Feedback por desempenho

🎨 Interface moderna com modal dinâmico

📊 Lógica de Correção

Cada quiz:

Define um objeto com respostas corretas

Percorre perguntas via loop

Valida se todas foram respondidas

Calcula percentual de acertos

Exibe resultado personalizado

Estrutura simples, robusta e fácil de expandir.
